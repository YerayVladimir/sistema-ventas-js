# Sistema de Gestión de Ventas

## Descripción
Proyecto JavaScript para procesar datos de ventas de una tienda online. Genera reportes y cálculos automáticos.

## Funcionalidades
- Filtrar ventas por categoría, precio y vendedor
- Calcular totales con descuentos
- Generar reportes de ventas por vendedor
- Identificar productos premium

## Tecnologías
- JavaScript (ES6+)
- Array methods: `.filter()`, `.map()`, `.reduce()`
- Arrow functions

## Cómo usar
1. Clonar el repositorio
2. Abrir `ventas.js` en tu editor
3. Ejecutar con Node.js: `node ventas.js`
